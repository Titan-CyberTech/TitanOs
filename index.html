Installation

Up and running
in minutes

01

Add ESP32 board URL

In Arduino IDE, open File → Preferences and add the Espressif board manager URL.

https://raw.githubusercontent.com/espressif/arduino-esp32/
gh-pages/package_esp32_index.json
02

Configure TFT_eSPI

Install TFT_eSPI via Library Manager, then uncomment the T-Display-S3 setup file.

#include <User_Setups/Setup206_LilyGo_T_Display_S3.h>
03

Set board parameters

Flash Size: 16MB · Partition: 16M Flash · PSRAM: OPI · Speed: 921600

04

Flash & boot

Connect via USB-C and click Upload. Hold BOOT + press RST if the port isn't detected.

Changelog

v2.0.0
2025 — Full rewrite
Added
  • Double-buffered rendering via TFT_eSprite — zero flicker on all screens
  • Persistent settings via ESP32 NVS — theme and brightness survive reboot
  • 5 color themes : Violet, Cyan, Orange, Green, Red
  • Topbar / Botbar UI on every screen with contextual button hints
  • Battery icon in topbar — color-coded (green / yellow / red) with % inside
  • Animated splash screen with loading bar
  • System Info — responsive 2×4 card grid layout
  • Snake — score, progressive speed, countdown, flashing game-over
  • Evil Portal — real-time credential display, capture counter, blink indicator
Fixed
  • Battery screen crash — button state not cleared caused stack overflow on return
  • Brightness not changing — 200ms debounce was swallowing button presses
  • Backlight pin corrected to GPIO38 (was GPIO15 in v1)
Changed
  • Complete architecture rewrite — one module per responsibility
  • ADC now uses esp_adc_cal with 8-sample averaging for accurate battery readings
  • All header guards replaced with #pragma once
